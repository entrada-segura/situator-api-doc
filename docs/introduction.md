# Situator API

> Base de conhecimento completa da API de integração com o sistema Situator.

## O que é uma API?

Uma API (Application Programming Interface) é uma Interface de Programação de Aplicação que atua como ferramenta de interação entre diferentes programas ao padronizar a comunicação, independentemente da tecnologia utilizada.

## O que é o Situator API?

O Situator API é uma interface que permite comunicação completa com o Situator via software externo. Benefícios:

- **Fácil Integração**: conecta diferentes sistemas e plataformas ao Situator
- **Automação**: reduz tarefas manuais repetitivas
- **Escalabilidade**: facilita o incremento de novas funcionalidades
- **Economia de tempo e custo**: utiliza serviços prontos

## Serviços Disponíveis

- Listar, inserir, alterar e apagar cadastros (contas, zonas, portas, pessoas, fotos, credenciais, permissões de acesso, veículos, previsões de visitas, divisões e áreas comuns)
- Gerar evento remoto
- Atendimento automático de ocorrências
- Webhooks (notificações de eventos)
- Relatórios de acessos e ocorrências
- Operações em painel de alarme (armar, desarmar, verificar status)
- Listagem de câmeras, imagens ao vivo e gravações
- Detalhes de dispositivos (fabricantes, modelos, periféricos, eventos)
