# Situator API

> Complete knowledge base for integrating with the Situator system.

## What is an API?

An API (Application Programming Interface) is an interface that acts as a tool for interaction between different programs by standardizing communication, regardless of the technology used.

## What is Situator API?

Situator API is an interface that enables complete communication with Situator via external software. Benefits:

- **Easy Integration**: connects different systems and platforms to Situator
- **Automation**: reduces repetitive manual tasks
- **Scalability**: facilitates adding new features
- **Time and cost savings**: uses ready-made services

## Available Services

- List, insert, update and delete records (accounts, zones, doors, people, photos, credentials, access permissions, vehicles, visit scheduling, divisions and common areas)
- Generate remote events
- Automatic occurrence handling
- Webhooks (event notifications)
- Access and occurrence reports
- Alarm panel operations (arm, disarm, check status)
- Camera listing, live images and recordings
- Device details (manufacturers, models, peripherals, events)
